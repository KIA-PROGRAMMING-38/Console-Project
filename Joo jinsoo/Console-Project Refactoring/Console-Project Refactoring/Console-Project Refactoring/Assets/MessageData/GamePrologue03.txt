

       게임 목표
       - 당신은 고용주에게 의뢰를 받고 단서를 쫓아 숨겨진 오피스에 찾아왔습니다
       - 당신은 이미 유력한 9개의 용의자, 살해도구, 살해동기를 추려냈습니다
       - 집안 구석구석을 확인해 단서를 얻어야 합니다
       - 시간이 없습니다. 당신에겐 4번의 추리 기회가 주어지게됩니다
       - 잘못된 추리를 입력할 경우 조사하다 놓쳤던 단서가 떠오릅니다

       스테이지 설명
       - (방의 이름)
         현재 당신의 위치를 나타냅니다
       - BB TT UU LL
         각 방으로 향하는 문을 나타냅니다
       - FF
         정답을 입력할 수 있는 지역입니다
         입력을 시도하거나 실패할 경우 기회가 차감합니다
       - P
         플레이어를 나타냅니다
         플레이어 아이콘의 색깔이 변하는 경우 상호작용이 가능합니다
       - 힌트
         상호작용으로 얻을 수 있는 사건 해결의 단서입니다
         힌트는 용의자, 살해도구, 살해동기 각각 하나씩 제공됩니다
         제공되는 순서는 무작위이며 플레이어의 추리가 필요합니다

       조작방법
       - ↑ ↓ ← →
         화살표로 플레이어를 이동시킬 수 있습니다
       - 키보드 R 입력
         플레이어의 색깔이 바뀐 경우, R키를 입력하여 단서를 찾아낼 수 있습니다

                    게임을 시작하려면 아무 키나 누르십시오
